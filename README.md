# CutFlow•CAD

材料取り作業に特化したシンプルな簡易CADソフト。
DXFファイルをフォルダごと取り込み、不要要素の削除・外径オフセット（加工代付加）・C面/開先指定を直感的に行い、加工用DXF/PDFとして書き出します。

---

## なぜ作るのか

製造現場の材料取り作業は、現在ベテランの手書きに依存しています。
属人化を解消し、若手スタッフでも迷わず使える共通基盤を提供するため、CutFlow•CADを開発します。

- 高齢スタッフの退職リスクへの備え
- 手書きとCADの世代間ギャップを埋める
- 設計部 / 製造部の役割整理を支える共通ツール
- システムを通じた情報共有とトレーサビリティ

ツールは「作業を巻き取る」のではなく、「**ベテランの判断を、若手が同じ手順で再現できる場**」を目指します。

---

## 主な機能

| # | 機能 | 概要 |
|---|---|---|
| 1 | **DXF取込・管理** | フォルダ単位で複数DXFを取り込み、画面下部のタブで瞬時に切替 |
| 2 | **外径自動検出** | LINE/ARC のトポロジ再構築で外径を抽出。失敗時は線クリックで手動指定 |
| 3 | **不要要素の削除** | 寸法線・バルーン・タップマーク・図枠を種類別/色別で一括削除、または個別選択 |
| 4 | **加工代の付加** | 外径全体に一律 2〜3mm、辺ごとの個別調整（E1〜En）にも対応 |
| 5 | **C面 / 開先** | 角・辺をクリックで指定、出力時に注記として記載 |
| 6 | **出力** | 材料取り用にクリーンアップしたDXF / PDFを書き出し |
| 7 | **拡張機能（将来）** | 寸法入力 / 線編集 / 穴追加 / 注記 / ブリッジ / ネスティング |

## 技術スタック

### フロントエンド
- **Vue 3** + **Vite** — SPA、社内PCのブラウザで動作
- **SVG** ベースのキャンバス — 編集操作は即時反映
- **IBM Plex Sans / Mono** + **Noto Sans JP** — 数値と和文の両立

### バックエンド
- **Python 3.12** + **FastAPI**
- **ezdxf** — DXF読み書き（AutoCAD 2010+ 対応）
- **Shapely** + **pyclipper (Clipper2)** — 外径オフセット計算
- **ReportLab** — PDF出力

### インフラ
- **Hostinger VPS** — Ubuntu 24.04 / 既存 `lineworks-x-ops` と隔離して同居
- **Docker Compose** — ローカルと本番で同一構成
- **Cloudflare Tunnel** — 本番公開時にIP固定不要・HTTPS自動

詳細は [`docs/ARCHITECTURE.md`](docs/ARCHITECTURE.md) を参照。
