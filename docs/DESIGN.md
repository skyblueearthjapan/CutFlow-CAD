## 2. ツール一覧（v3 で確定済み）

### 【編集】グループ（実装対象）

#### 1. 外形 (`outer`) — キー `1`
- **目的**: 外径を自動検出して可視化、必要なら手動修正
- **動作**:
  - 自動検出（トポロジ再構築）で閉ループを抽出
  - 検出成功 → 外径をシアンで強調、信頼度% を表示
  - 検出失敗 → アンバー警告バナーを表示、最有力候補を点滅
  - 「線を手動指定」で線をクリックチェーン選択
- **インスペクタ表示**:
  - 外径構成（CLOSED / segments / LINE n / ARC m）
  - 外周長、面積
  - 検出ピル（自動 92% など）
  - 警告ストリップ（閉ループ未確認件数）

#### 2. 削除 (`delete`) — キー `2`
- **目的**: DXF由来の製図情報をクリーンアップ
- **動作**:
  - 種類別チェックボックスでON/OFF（寸法線・バルーン・タップ穴マーク・図枠）
  - 各行に件数バッジ
  - キャンバスでクリックして個別選択も可能
  - 「削除後プレビュー」で残るエンティティ数・推定切断長を表示
- **削除判定ルール（レイヤー未設定の前提）**:
  - `DIMENSION` エンティティ → 寸法線として確実に検出
  - `INSERT` ブロック + 円形（小） → タップマーク候補
  - `INSERT` ブロック + 円形 + 数字テキスト併走 → バルーン候補
  - 外周を囲む長方形 + 表題欄テキスト → 図枠候補
  - 色（DXFカラーインデックス）も補助的に利用

#### 3. 加工代 (`offset`) — キー `3`
- **目的**: 外径から外側にオフセットして材料取り寸法を確保
- **動作**:
  - デフォルト値（2.0 または 3.0 mm）を全周に一律適用
  - 辺ごとに個別調整可能（E1〜En）
  - 角の処理（円弧連結 / 鋭角延長）を選択
  - プレビューを点線シアンで重ね表示
- **インスペクタ表示**:
  - デフォルト num-step + 単位
  - 辺ごとリスト（インデックス / 種別 / 長さ / 現在値）
  - 適用後の外周長・板取り寸法・材料効率%

#### 4. C面 / 開先 (`chamfer`) — キー `4`
- **目的**: 角・辺に面取り/開先を指定（出力時の注記）
- **動作**:
  - 角コーナーチップ（4隅 + 必要なら任意箇所）で指定
  - C面サイズ / 開先角度を設定
  - 指定箇所はパープルで可視化
- **インスペクタ表示**:
  - C面サイズ、開先角度の num-step
  - 指定済み角のチップ一覧
  - DXF出力時の注記プレビュー

### 【作図】グループ（Phase 2 以降）

#### 5. 寸法 (`dim`) — キー `5`
- 2点クリックで寸法線追加、`D` キーで2点間モード
- ISO標準・小数桁・矢印サイズを設定

#### 6. 線編集 (`edit`) — キー `6`
- 頂点・線分のドラッグ編集、スナップ・グリッド吸着
- 直交モード（Shift）

#### 7. 穴追加 (`hole`) — キー `7`
- 任意位置にクリックで穴を追加（φ指定）
- 連続配置 / 整列パターン（`A` キー）

#### 8. 注記 (`note`) — キー `8`
- 加工指示テキストの追加（面粗さ・溶接・一般）
- フォントは isocp 2.5mm 等

### 【出力】グループ

#### 9. ブリッジ (`bridge`) — キー `9`
- 切断時の保持タブを外径に残す（部品脱落防止）
- ブリッジ幅・推奨個数（重量より算出）・等間隔配置

#### ネスティング (Phase 5 — 実装済)
- 複数部品を 1 枚の板に Bottom-Left-Fill (BLF) で最適配置
- POST `/api/session/{sid}/nest` で非同期ジョブ投入、`GET /api/jobs/{job_id}` で進捗、`GET /api/jobs/{job_id}/result` で結果取得
- シート別 DXF は `GET /api/jobs/{job_id}/result/sheets/{idx}/export?format=dxf`
- 真の No-Fit Polygon (凹形対応) は Phase 6 で追加予定

## 4. 入出力仕様

### 入力
- DXFファイル（AutoCAD 2010+ / AC1024〜）
- **フォルダ単位**で複数DXF同時アップロード
- 組立図 (`*組立図*`) は自動除外、バラ図のみ取り込み
- 最大ファイルサイズ: 1ファイルあたり 5MB、フォルダあたり 50ファイル

### 出力
- **DXF**: 材料取り用にクリーンアップ済み（製図要素削除・加工代適用・C面注記）
- **PDF**: 紙印刷用、図枠なし / 1ページ1部品
- 出力時に**材料取り専用枠**（後日定義）を選択可能。当面は「枠なし」を初期値

### データ保存
- サーバには**一時保存のみ**（セッション24時間で自動削除）
- 編集完了 → ブラウザで自動ダウンロード
- 認証なし（社内ネットワーク前提）

## 5. 削除判定アルゴリズム（重要）

レイヤー名が**数字のみ**（御社サンプルで確認済み）のため、色 + エンティティ種別 + ジオメトリ特徴で判定:

```
寸法線      = entity_type == "DIMENSION"
              OR (LINE + 並行短線 + 中央にMTEXT/TEXT)
バルーン    = INSERT + 内側に CIRCLE + 中央にTEXT(数字1〜2桁)
              OR 単独の小円 + 引出線 + 番号テキスト
タップ      = INSERT(タップブロック慣用名: TAP, M*, ねじ*)
              OR 二重円(外側破線) + Mn テキスト併走
図枠        = 外周を囲む最大の矩形 LWPOLYLINE
              + 右下/左下に表題欄(罫線で区切られた矩形群)
              + 図番テキスト
```

検出は全て**ユーザー確認・取り消し可能**にし、誤検出のリスクを下げる。

## 6. 外径自動検出アルゴリズム

```
[STEP 1] エンティティ収集
  対象: LINE / ARC / LWPOLYLINE / POLYLINE / CIRCLE
  除外: 寸法線(DIMENSION)・バルーン候補・タップ候補

[STEP 2] 端点マージ
  許容誤差 0.01mm で全エンティティの端点を統合
  ノードグラフを構築

[STEP 3] 最外周ループ抽出
  グラフの外周を時計回り / 反時計回りに探索
  最大面積の閉ループを外径候補に

[STEP 4] 信頼度算出
  - ループの完結性（端点が完全に閉じているか）
  - 面積比（外周の妥当性）
  - 内側に他要素があるか（穴・タップ等は内側にあるべき）

[STEP 5] 結果処理
  信頼度 ≥ 80% → 自動確定（シアン強調）
  信頼度 50〜80% → 警告バナー + 確認を促す
  信頼度 < 50% → 失敗、手動指定モードへ
```

手動指定: 外径を構成する線をクリックチェーン → 「閉じる」ボタンで最後の線と最初の線を自動接続。
